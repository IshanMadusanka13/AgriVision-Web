What We Built

Three AI-Powered Modules

AgriVision unifies plant health monitoring, produce quality grading, and adaptive planting advisory into one mobile-friendly ecosystem for Scotch Bonnet farmers.

Plant Monitoring & Recommendation

Detect diseases early, track plant growth with ArUco marker-based height measurement, and receive custom growing recommendations based on environmental data.

Quality Grading & Visual Sorting

Automatically classify batches of up to 50 Scotch Bonnet peppers into Grades A–D using color and surface blemish detection via YOLOv8 segmentation.

Adaptive Layout & Advisory System

Map field boundaries via GPS, classify soil fertility using Random Forest, compute optimal plant spacing and density, and predict expected yield.

Domain

Literature survey, research gap, problem statement, objectives, methodology, and technologies used.

Background

Literature Survey

Recent advances in deep learning and computer vision have dramatically improved agricultural monitoring. This research builds on that foundation to address gaps specific to Scotch Bonnet cultivation.

Plant Disease Detection

CNNs including YOLO-family models and U-Net segmentation have been widely applied to leaf-level disease classification. However, most existing systems focus only on classification and lack severity estimation or stage progression analysis.

Growth Monitoring

Image-based phenotyping with deep learning can extract plant traits like height and structure. Most systems operate under controlled conditions and are rarely integrated into real farming environments.

Quality Grading

Deep learning models are used to sort fruits by size, shape, color, and defects. Limited research specifically targets color and blemish-based grading for crops like Scotch Bonnet peppers.

Precision Agriculture

Random Forest classifiers excel at soil fertility classification using NPK, pH and moisture features. Variable spacing and plant density optimization improve field performance and sustainability.

Gap Identified

Research Gap

Identified Gaps

No existing system integrates disease detection, growth monitoring, quality grading, and planting advisory into a single platform. Disease severity and comparative growth benchmarking against peer plants are largely absent from prior work.

Core Problem

Research Problem

How can AI-based tools improve Scotch Bonnet farming?

Small-scale Scotch Bonnet farmers in Sri Lanka rely on manual observation, leading to late disease detection, inconsistent quality assessment, and poor planting decisions. A unified, mobile-accessible AI system can bridge these gaps.

Goals

Research Objectives

Objective 01

Build an image-based pipeline for disease detection and severity estimation on Scotch Bonnet leaves using EfficientNetB0 and U-Net.

Objective 02

Develop a real-time growth monitoring module using ArUco markers and YOLOv8 to track plant height and growth stage over time.

Objective 03

Create an automated quality grading system classifying peppers into four grades based on color and blemish analysis via YOLOv8 segmentation.

Objective 04

Implement an adaptive planting advisory using GPS field mapping, Random Forest soil classification, and yield prediction for optimal plant density.

How It Works

Methodology

A systematic approach integrating mobile image capture, deep learning pipelines, and rule-based advisory for real-world farming conditions.

Disease Detection Pipeline

YOLOv8 detects and isolates individual leaves → EfficientNetB0 classifies disease type (Healthy / Cercospora / Bacterial) → U-Net performs pixel-level segmentation to compute severity ratio: Diseased Area / Total Leaf Area × 100.

Growth Monitoring

ArUco markers (5×5 cm) provide a physical reference scale. Plant height is derived from pixel measurements. Growth rate = (H_Current − H_Previous) / Δt, benchmarked against well-performing plants in the same growth stage.

Quality Grading

YOLOv8-seg detects and segments individual peppers in batch images (up to 50 per image). Grading uses color (green/yellow/orange-red) and surface defect detection → Grade A, B, C, or D.

Adaptive Layout

GPS boundary traversal → polygon area calculation. Soil vector [N, P, K, pH, Moisture] → Random Forest classifies fertility (Rich/Medium/Poor) → adaptive intra-row spacing (S_i) → Plant density D = 1/(S_i × 0.75). Yield predicted via RF regression trained on 500 records.
